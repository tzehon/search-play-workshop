## Only make searchable a specific subset of fields

<Playground url="https://search-playground.corp.mongodb.com/tools/code-sandbox/snapshots/67b5ccc0828ad15116bcce21">ğŸ”— Playground link: a dynamic string field is automatically searchable</Playground>

Make it so only fields under the `metadata` structure are indexed (and thus this query would then fail to find any matching documents). This is an exercise where no matching documents (an empty array `[]`) is the goal, rather than matching on a field that should not be indexed.

<details>
  <summary>Solution</summary>
  <div>
    Turn off top-level dynamic mapping, and only index the `metadata` sub-structure.
    ```js
    {
      "mappings": {
        "dynamic": false,
        "fields": {
          "metadata": [
            {
              "type": "document",
              "dynamic": true
            }
          ]
        }
      }
    }
    ```
