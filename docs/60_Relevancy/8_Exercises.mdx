## project the score details

<Playground url="https://search-playground.mongodb.com/tools/code-sandbox/snapshots/67d2ee9b643f61797f19d33c">ğŸ”— Playground link: add the score details into the results</Playground>

Enable the score details and project them into the results.

<details>
  <summary>Solution</summary>
  <div>
    Enable the score details with this setting in `$search`:
    ```js
    scoreDetails: true
    ```

    Include the score details by adding this to `$addFields`:
    ```js
    "scoreDetails": {
      "$meta": "searchScoreDetails"
    }
    ```
    <Playground url="https://search-playground.mongodb.com/tools/code-sandbox/snapshots/67d2eec2808223b17a90bd78">ğŸ”— solution</Playground>
  </div>
</details>
